◆ generateImageUrls()

std::vector< std::string > generateImageUrls ( const std::string & apiKey,
int numimages )

Generates a list of public domain image URL from public domain image repositories on the Web.

Google Gemini is utilized to generate the list of image URLs in a two-step process with two prompts. The first one generates URLs directly pointing to a valid image file in either JPEG or PNG format. The second one is used to extract only the list of URLs from the output of the first prompt as there is no guarantee that the first prompt generates only the list of image URLs

Parameters
apiKeyAPI key to Google Gemini
numimagesNumber of images to generate
Returns
List of image URLs

A callback for libcurl.

When libcurl is used to to perform an HTTP request, it needs to know how to handle the incoming data (the response body). This function is registered with CURLOPT_WRITEFUNCTION, and each time libcurl receives a block of data, it calls this function.

Parameters
contentsPointer to the block of data received from the HTTP response
sizeThe size (in bytes) of each data element
num_dataThe number of data elements
userpPointer to a string that will hold the response
Returns
Number of bytes handled (size * num_data)
